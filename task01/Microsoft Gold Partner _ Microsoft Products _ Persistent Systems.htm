Microsoft

Microsoft Practice Overview

Persistent Systems is a Microsoft Gold Certified partner and has extensive experience in building customer solutions using Microsoft products. Persistent has contributed to the development of the Microsoft product suite and helped customers build superior quality applications and solutions across several Microsoft product lines, including Windows Azure, SharePoint, SQL Server and Windows Phone 8.

As a service provider, Persistent focuses on BI & Analytics, Cloud Computing, Mobility and Social and can leverage the power of Microsoft technologies to help companies from a variety of industries, including Technology, Telecom, Life Sciences, CPG, Banking & Financial Services and Healthcare.

"Persistent has proven to be a core SQL Server vendor and partner. They continue to take on challenging projects and deliver them successfully for the SQL group – like the Best Practices tool for SQL and cutting edge proof of concepts for the next SQL release in addition to sustenance projects."

Dave Campbell

GM, Technology Fellow SQL Server News
